Mantis runs on Linux, macOS, and Windows. There are three main methods to run the Mantis client:

* using prebuilt-binaries (recommended)
* using binary archive
* building from source (follow these instructions to build a client from source).

Install on [Linux](../install/linux), [macOS](../install/macos) or [Windows](../install/windows).
