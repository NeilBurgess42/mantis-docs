## Supported platforms

Windows 8.1, Windows 10 (Only 64-bit Windows is supported)  
MacOS 10.13, MacOS 10.14, MacOS 10.15  
Linux - Aimed at all Linux distributions - Tested against:  
Ubuntu 18.04 LTS  
Ubuntu 20.04 LTS  
Fedora 28  


## Recommended minimum hardware requirements

64-bit dual core processors  
4 GB of RAM  
at least 8GB disk space for IOHK and Mordor  
At least 50gb for FastSync  
at least 1.2TB disk space for Mainnet  
Broadband Internet connection  


## Advanced Users

Advanced Users may want to launch the Mantis wallet through command line, in order to pass it extra configuration options with Environment Variables.

Download and run the wallet's packaged binary.

```
cd Applications
wget https://github.com/input-output-hk/mantis-wallet/releases/download/untagged-2aa879d1ad64a6b57e5e/Mantis-Wallet-0.1.0.AppImage
```

Run it on by executing the wallet binary.

```
./Mantis-Wallet-0.1.0.AppImage # replace with current version and correct OS option
```

Once the Mantis splash screen is finished loading, visit the `Settings` menu and ensure you are connected to the correct Network.

> Note: The wallet runs a Mantis client behind the scenes and will need to synchronize with the network initially. This might take several hours, depending on which network it connects to.
