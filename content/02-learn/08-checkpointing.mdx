Checkpointing is a system with the help of which the Sagano testnet provides the Ethereum Classic blockchain with 51% attack resistance. It protects the network from attacks ensuring confidence in system security. 

For more details, please follow the link to Ethereum Classic Improvement Proposals ([ECIPs](https://github.com/ethereumclassic/ECIPs/blob/master/_specs/ecip-1097.md)), and also read the [white paper](https://dimkarakostas.com/checkpointed_blockchains.pdf).
